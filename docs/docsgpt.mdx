# DocsGPT

DocsGPT is an AI-powered chatbot designed to help users easily access and understand documentation. Instead of searching through long and complex documentation, you can simply ask the chatbot about the documentation you need.

## **What Does DocsGPT Do?**
- 📄 **Retrieve Documentation** – Retrieve information from the documentation.
- 🔍 **Ask Questions** – Get AI-generated answers to specific questions about the documentation.
- 🌍 **Compare Differences** – See differences in documentation between different versions.
- ✨ **Summarised Insights** – Get key takeaways from lengthy documentation.

DocsGPT makes navigating documentation effortless. Whether you're a developer, researcher, or business, our goal is to simplify access to important documentation.
